# PoE Build Pricer

**Live:** https://eazzzymax.github.io/buildPricer/

A handy tool to manually assemble a list of items for a future Path of Exile build and estimate its total cost before you start putting it together.

Удобный инструмент, чтобы вручную собрать список предметов будущего билда в Path of Exile и прикинуть его суммарную стоимость ещё до того, как начнёшь его собирать.
